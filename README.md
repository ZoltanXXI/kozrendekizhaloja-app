# kozrendekizhaloja-app
Közrendek Ízhálója egy interaktív Streamlit webalkalmazás, amely a XVII. századi magyar "közrendi" konyha alapanyagait és receptjeit hálózati szemléletben mutatja be. AI-alapú keresővel, vizuális gráffal és archaizáló stílusú, automatikusan generált receptekkel segíti a történeti gasztronómia felfedezését.
